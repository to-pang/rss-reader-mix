# RSS-Reader-Mix

基于 [rss-reader](https://github.com/srcrs/rss-reader) 原项目的样式改进版，旨在提供更个性化的 RSS 阅读体验。

## 🌟 改进特性
相比于原版，本项目（Mix 版）进行了以下优化：
- **自定义外观**：支持在配置文件中直接修改网页标题（webTitle）和描述（webDes）。
- **布局优化**：新增 `listHeight` 参数，可自由控制列表高度（建议源较多时设为 300-600）。
- **样式精调**：微调了前端 UI 样式，视觉更加美观。
- **原有优势**：保留了原版镜像极小（约 20MB）、响应式布局、SEO 友好及多链接支持等特性。
